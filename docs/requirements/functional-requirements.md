# 機能要件

## 1. 対象範囲
Phase1では、開発者本人を主な利用者とし、
現在の資産状況の把握と、一人暮らし開始可否の判断を支援する。

提供する主な機能は、以下とする。

- 資産口座の管理
- 月末時点の資産残高の記録
- 金融商品別の残高の記録
- CSVによる月末残高の取り込み
- 総資産、口座別資産、商品別資産の推移表示
- 手取り収入の記録
- 一人暮らし開始可否のシミュレーション

## 2. 資産口座管理
### 2.1 資産口座・保管場所の登録
利用者は、資産を保有している口座、制度または保管場所を登録できる。
登録対象の例は以下とする。
- 家にある現金
- 財布
- 銀行口座
- 証券口座
- iDeCo
- 企業型DC

登録時には、少なくとも以下を設定できる。
- 口座・保管場所名
- 種別
- 金融機関または運営機関
- 利用可能資産として扱うか
- 残高の記録単位
- 有効または無効の状態

残高の記録単位は、以下から選択できる。
- 口座単位
- 商品単位

口座単位を選択した場合、
当該口座または保管場所の月末残高を直接登録する。

商品単位を選択した場合、
当該口座に属する保有商品の月末評価額を登録し、
その合計額を口座の月末残高として扱う。

### 2.2 資産口座・保管場所の編集
利用者は、登録済みの資産口座・保管場所情報を編集できる。
編集可能な項目は以下とする。

- 口座・保管場所名
- 種別
- 金融機関または運営機関
- 利用可能資産区分
- 残高の記録単位
- 有効状態

残高履歴が登録済みの資産口座について、
残高の記録単位を変更する場合は、
過去の集計結果や資産推移に影響する可能性があるため、
変更前に確認を表示する。

### 2.3 資産口座・保管場所の無効化
過去の残高履歴を保持したまま利用を終了できるよう、
登録済みの資産口座・保管場所を削除せず無効化できる。

無効化した資産口座・保管場所は、
新しい月末残高の登録対象には表示しない。

ただし、過去の月末残高や資産推移には引き続き表示する。

### 2.4 月末残高の重複計上防止
同一の資産口座について、
口座単位の月末残高と商品単位の月末評価額を、
同じ対象年月の総資産へ重複して加算しないものとする。

残高の記録単位が商品単位の場合、
保有商品の月末評価額の合計を口座残高として扱う。

## 3. 月末資産残高管理
### 3.1 月末資産残高の登録
利用者は、登録済みの資産口座または保管場所について、
対象月の月末時点における残高を登録できる。

登録方法は以下とする。
- 手入力
- CSVインポート

登録時には、少なくとも以下を設定できる。
- 対象年月
- 資産口座・保管場所
- 月末残高
- 備考

同一の資産口座・保管場所について、
同一対象年月の月末残高を重複して登録できないものとする。

### 3.2 月末資産残高の編集
利用者は、登録済みの月末資産残高を編集できる。
編集可能な項目は以下とする。
- 対象年月
- 資産口座・保管場所
- 月末残高
- 備考

### 3.3 月末資産残高の削除
利用者は、誤って登録した月末資産残高を削除できる。
ただし、削除前に確認を表示する。

### 3.4 CSVによる一括登録
利用者は、複数の月末資産残高をCSVから一括登録できる。
CSVインポートおよびCSV出力の詳細は、
「5. CSVインポート」に定義する。

## 4. 保有商品管理

### 4.1 保有商品の登録
利用者は、証券口座、iDeCo、企業型DCなどの資産口座に対して、
保有している金融商品を登録できる。

登録時には、少なくとも以下を設定できる。
- 資産口座
- 商品名・銘柄名
- 商品種別
- 備考
- 有効または無効の状態

商品種別の例は以下とする。
- 投資信託
- 国内株式
- 外国株式
- 預金商品
- その他

### 4.2 保有商品の編集
利用者は、登録済みの保有商品情報を編集できる。
編集可能な項目は以下とする。

- 資産口座
- 商品名・銘柄名
- 商品種別
- 備考
- 有効状態

### 4.3 保有商品の無効化
過去の商品別月末残高履歴を保持するため、
利用を終了または売却した保有商品は削除せず無効化できる。

無効化された保有商品は、新しい月末残高の登録対象には表示しない。
過去の資産推移には引き続き表示する。

### 4.4 商品別月末残高の登録

利用者は、登録済みの保有商品について、
対象月の月末時点における評価額を登録できる。

登録時には、少なくとも以下を設定できる。
- 対象年月
- 保有商品
- 月末評価額
- 備考

同一の保有商品について、
同一対象年月の月末評価額を重複して登録できないものとする。

### 4.5 商品別月末残高の編集・削除
利用者は、誤って登録した商品別月末残高を編集または削除できる。
削除前には確認を表示する。

### 4.6 資産口座残高への集計
資産口座ごとに、残高の記録方法を以下から選択できる。
- 口座単位で月末残高を記録する
- 商品単位で月末評価額を記録する

商品単位で記録する資産口座については、
その口座に属する商品別月末評価額の合計を、
当該資産口座の月末残高として扱う。

口座単位の月末残高と商品別月末評価額を
重複して総資産へ加算しないものとする。
